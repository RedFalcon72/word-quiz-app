# Word Quiz App

## Overview

Word Quiz App is a simple browser-based vocabulary learning application.

Users can create their own word books and practice vocabulary by answering whether they know the word or not.

This project was built to practice **React , Tailwind CSS**,and to develop an understanding of **component-based design and state management**..

## Live Demo:https://redfalcon72.github.io/word-quiz-app/

## Tech Stack

- TypeScript
- React
- Tailwind CSS

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/RedFalcon72/word-quiz-app
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
